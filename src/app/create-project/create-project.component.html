Creating your first Angular project:

Prerequisites:

  1. Install Node.js and npm
    • Download and Install Node.js
    • Verify installations by: 
      node --version
      npm --version
  2. Install Angular CLI
    • Open a command prompt or temminal in your project directory
    • Run the following command: 
      npm install -g @angular/cli
    • Verify installation: 
      ng --version

Steps to Create new Angular Project:

  1. Open terminal or command prompt
    • Navigate to the directory where you want to create your project
  2. Create a new project
    • Run the following command: 
      ng new my-app
    • Replace "my-app" with the name of your project
  3. Answer the following Configuration Questions
    • Would you like to add Angular routing?(y/n)
      • Choose 'y' if you would like to add Angular routing
    • Which stylesheet format would you like to use?
      • Choose from CSS, SCSS or LESS(Defualt is CSS)
  4. Wait for the project to build
    • Angular CLI will install required dependencies
  5. Navigate to your project directory 
    cd my-app
  6. Serve the project
    • Run your project: 
      ng serve
