Let's Learn Angular

What is Angular?

Angular is a platform and framework for building single-page client applications using HTML and TypeScript. It implements core and optional functionality as a set of TypeScript libraries that you import into your applications.

Angular CLI:

Angular CLI is a command-line interface (CLI) for Angular. Angular CLI is a powerful tool that greatly simplifies the process of developing, testing, and deploying Angular applications. It is designed to streamline the development workflow for Angular projects. It automates many common tasks, enforces best practices, and helps maintain consistency across projects.

Angular CLI, node.js and npm:

Understanding the relationship between Angular CLI, Node.js, and npm is crucial for Angular development.

  1. Node.js Dependency:
    • JavaScript Runtime:
      • Angular CLI is built on Node.js, which provides a JavaScript runtime environment outside the browser. This allows Angular CLI to execute JavaScript code on your machine for tasks like project scaffolding, building, and serving.
  2. npm:
    • npm is used to manage dependencies for your Angular project. It downloads and installs all the necessary libraries and tools that Angular needs.
    • Global installation of npm packages: 
      npm install -g @angular/cli

Creating your first Angular project:

  1. Install Node.js and npm
    • Download and Install from node.js
    • Verify installations by: 
      node --version
      npm --version
  2. Install Angular CLI:
    • Open a command prompt or temminal in your project directory
    • Run the following command: 
      npm install -g @angular/cli
    • Verify installation: 
      ng --version
